# Client API

## Initialize the Client

The DXOS Client can be created and initialized programatically using the `@dxos/client` package.

```js
import { Client } from '@dxos/client';

export const Test = async () => {
  const client = new Client({
    ...
  });

  await client.initialize();
  const profile = await client.halo.createProfile({ usename: 'tester' });
  console.log(profile.username);
};
```

## Using React

The `@dxos/react-client` package provides hooks to craete and configure the Client.

```jsx
import React, { useEffect } from 'react';
import { ClientProvider, useClient } from '@dxos/react-client';

const App = () => {
  const client = useClient();
  const profile = useProfile();
  useEffect(() => {
    setImmediate(async () => {
      await client.halo.createProfile({ usename: 'tester' });
    });
  }, []);

  return (
    <span>Welcome ${profile.username}</span>
  );
};

export const Test = () => {
  const config = {
    ...
  };

  return (
    <ClientProvider config={config}>
      <App />
    </ClientProvider>
  );
};
```
